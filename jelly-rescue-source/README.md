# 젤리구조대 — GitHub Edition

한 컴퓨터에서 두 사람이 함께 플레이하는 협동 퍼즐 플랫폼 게임입니다. 두 젤리가 상자를 옮기고 발판을 눌러 구조 문을 연 뒤, 함께 구조 구역에 들어가면 성공합니다. 현재 열 개의 협동 스테이지가 포함되어 있습니다.

## 실행 방법

Node.js 20 이상이 필요합니다. 이 폴더에서 명령 프롬프트를 열고 아래 두 줄을 차례로 실행하세요.

```bash
npm install
npm run dev
```

화면에 나타나는 `http://localhost:5173` 주소를 크롬에서 열면 됩니다. 설치 주소가 다른 곳으로 잡히는 문제를 막기 위해 프로젝트의 `.npmrc`는 공식 npm 저장소만 사용하도록 설정되어 있습니다.

## 조작

| 플레이어 | 이동 | 점프 |
|---|---|---|
| 파란 젤리 (P1) | `A`, `D` | `W` |
| 분홍 젤리 (P2) | `←`, `→` | `↑` |

- `R`: 현재 스테이지 다시 시작
- `ESC`: 시작 화면으로 돌아가기
- 시작 화면의 숫자 `1`, `2`, `3`: 원하는 스테이지 바로 시작

## 스테이지 공략 목표

### 스테이지 1

1. 상자를 높은 발판 왼쪽의 지지 기둥 앞까지 밉니다.
2. 한 젤리는 바닥 발판, 다른 젤리는 위쪽 발판을 동시에 누릅니다.
3. 문이 열리면 두 젤리가 오른쪽 구조 구역 안에 함께 들어갑니다.

### 스테이지 2

1. 첫 번째 상자를 왼쪽 상자 발판까지 옮겨 계단으로도 사용합니다.
2. 위쪽의 두 번째 상자를 아래로 떨어뜨려 오른쪽 상자 발판까지 옮깁니다.
3. 두 상자가 발판을 누르면 열린 문으로 함께 탈출합니다.

### 스테이지 3

1. 두 젤리가 보라색 무거운 상자의 같은 쪽에 섭니다.
2. 두 플레이어가 같은 방향키를 동시에 눌러 상자를 함께 밉니다.
3. 무거운 상자를 전용 발판에 올린 뒤 구조 구역으로 탈출합니다.

### 스테이지 4~10

- 4단계: 단서를 해석해 세 신호를 올바른 순서로 작동
- 5단계: 친구를 밟고 올라가는 협동 점프
- 6단계: 한 명이 전원을 유지해야 움직이는 승강기
- 7단계: 한 명이 문을 유지하고 다른 한 명이 안쪽 잠금을 해제하는 릴레이
- 8단계: 제한 시간과 이동 거리를 고려한 상자 역할 분담
- 9단계: 단서에 따라 무거운 상자와 가벼운 상자를 순서대로 배치
- 10단계: 무거운 상자가 승강기 전원도 담당하는 종합 작전

4단계부터는 화면의 목표가 정답을 직접 알려주지 않습니다. 두 플레이어가 중앙의 단서와 장치 이름을 살펴보고 역할과 행동 순서를 먼저 상의하도록 설계되어 있습니다.

## 배포용 빌드

```bash
npm run build
```

완성 파일은 `dist` 폴더에 만들어집니다.

## GitHub Pages 공개

이 프로젝트에는 자동 배포 설정이 포함되어 있습니다.

1. GitHub에서 새 저장소를 만들고 이 폴더의 파일을 올립니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. **Source**를 **GitHub Actions**로 선택합니다.
4. `main` 브랜치에 코드를 올리면 Actions가 게임을 자동 공개합니다.

## 프로젝트 구조

```text
src/
├─ audio/       효과음 생성
├─ levels/      스테이지 배치 정보
├─ objects/     젤리, 상자, 발판, 문
└─ scenes/      시작, 게임, 결과 화면
```

현재 버전은 외부 이미지와 음원 없이 실행됩니다. 모든 그래픽과 효과음은 게임 안에서 생성되므로 저장소를 복제한 뒤 바로 실행할 수 있습니다.
